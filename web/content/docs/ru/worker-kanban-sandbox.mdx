## Как устроен пример

Эта страница показывает полный worker-based сценарий без моков. Docs-страница держит host DOM и pointer hit-testing, а remote Vue-приложение с состоянием канбан-доски работает внутри выделенного `Web Worker`.

Пример состоит из трех частей:

- host runtime, который монтирует `HostedTree` в область preview;
- remote Worker, который рендерит колонки, карточки и обрабатывает `RemoteSortableEvent`;
- live snapshot под доской, который показывает текущее remote-state после каждого drop.

## Как взаимодействовать

- Захватывайте карточки только за кнопку `Drag`.
- Наводите карточку между элементами в колонке для reorder или в другую колонку для переноса.
- Используйте `Сбросить доску`, чтобы вернуть исходное состояние примера.
