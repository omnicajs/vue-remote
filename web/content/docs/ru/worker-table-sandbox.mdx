## Как устроен пример

Эта страница показывает worker-based таблицу без моков. Docs-страница держит host DOM и pointer hit-testing, а remote Vue-приложение и порядок строк работают внутри выделенного `Web Worker`.

Пример состоит из трех частей:

- host runtime, который монтирует `HostedTree` в область preview;
- remote Worker, который рендерит строки таблицы и обрабатывает `RemoteSortableEvent`;
- live snapshot под таблицей, который показывает текущее remote-state после каждого drop.

## Как взаимодействовать

- Захватывайте строки только за кнопку `Drag` в первом столбце.
- Перемещайте строку выше или ниже другой строки, чтобы менять порядок.
- Используйте `Сбросить таблицу`, чтобы вернуть исходную очередь.
