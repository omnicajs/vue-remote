## 这个示例由哪些部分组成

这个页面展示的是完整的 Worker 嵌套文件树场景，而不是模拟预览。docs 页面负责 host DOM 和指针命中测试，remote Vue app 则在专用 `Web Worker` 中保存树结构、inline 编辑状态和 drag 逻辑。

这个示例包含三部分：

- 把 `HostedTree` 挂到预览区域中的 host runtime；
- 渲染文件夹、文件、inline 操作并处理 `RemoteSortableEvent` 的 remote Worker 应用；
- 位于树下方的实时 snapshot，用来显示每次编辑和 drop 之后的当前嵌套 remote state。

## 如何交互

- 通过 handle 拖拽节点来重排同级节点或把它们移动到其他文件夹。
- 直接 inline 重命名项目、添加文件或文件夹，并从当前树中删除节点。
- 使用 `重置树` 可以恢复初始 workspace 结构。
