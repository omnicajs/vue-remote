## この例の構成

このページはモックではなく、完全な Worker ベースの構成をそのまま見せています。docs ページが host DOM と pointer hit-testing を担当し、remote Vue app とカンバンの状態は専用 `Web Worker` の中で動きます。

この例は次の 3 つの要素で構成されています。

- preview 領域に `HostedTree` をマウントする host runtime
- レーンとカードを描画し、`RemoteSortableEvent` を処理する remote Worker app
- drop のたびに現在の remote state を表示する、ボード下の live snapshot

## 使い方

- カードは `Drag` ハンドルからだけつかみます。
- 同じ列のカード間に移動すると並べ替え、別の列へ移動するとカードを移送できます。
- 初期状態に戻したいときは `ボードをリセット` を使ってください。
